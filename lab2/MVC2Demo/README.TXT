Has framework Jar file in Web App Libraries
The Jar file has source code included.

Here is how to attach the source code to the classes:

Right click on project -> Properties ->Java Build Path 
- expand Web App Libraries
- expand MVC2Framework
- click on Source attachment
- choose "edit" button on right
- Navigate to MVC2Demo->WebContent -> WEB-INF -> lib -> Mvc2Framework.jar

CLICK OK...
